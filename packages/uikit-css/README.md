# @freecodecamp/uikit-css

Design tokens, component CSS, fonts, and brand assets for freeCodeCamp UIKit.

## Install

```bash
pnpm add @freecodecamp/uikit-css
```

## Use

```ts
import '@freecodecamp/uikit-css';
```

Direct CSS subpath imports are also available:

```ts
import '@freecodecamp/uikit-css/tokens.css';
import '@freecodecamp/uikit-css/components.css';
```

## Exports

- `@freecodecamp/uikit-css`
- `@freecodecamp/uikit-css/index.css`
- `@freecodecamp/uikit-css/tokens.css`
- `@freecodecamp/uikit-css/components.css`
- `@freecodecamp/uikit-css/fonts/*`
- `@freecodecamp/uikit-css/brand/*`
- `@freecodecamp/uikit-css/package.json`

This package ships source CSS directly. There is no compile step for package
consumers.
