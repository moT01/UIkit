The CDN bundle is a single `<link>` for CSS and one or two `<script>`
tags for the SVG sprite + the vanilla-JS adapters. Tree-shaking is
unavailable, but for prototypes, learner sandboxes, and quick demos,
the trade-off is fine.

## Tag layout

```html
<link
  rel="stylesheet"
  href="https://cdn.freecodecamp.org/uikit/styles.min.css"
/>

<button class="btn btn--cta">Start</button>

<!-- IIFE wires modal, dropdown, tabs, toggle, tooltip -->
<script src="https://cdn.freecodecamp.org/uikit/uikit.global.js" defer></script>
```

## Mounting the sprite

Icons are referenced via `<svg><use href="#icon-name" /></svg>`.
Load the sprite once into the DOM and the rest of the page gets all
icons for free.

```html
<!-- one-shot fetch + injection, before any icon usage -->
<script>
  fetch('https://cdn.freecodecamp.org/uikit/sprite.svg')
    .then(r => r.text())
    .then(svg => {
      const wrapper = document.createElement('div');
      wrapper.style.display = 'none';
      wrapper.innerHTML = svg;
      document.body.prepend(wrapper);
    });
</script>
```

## IIFE init contract

`uikit.global.js` is a side-effect bundle: importing it (or
`<script src=…>`-ing it) hooks every `data-*` attribute the UIKit
recognises. No explicit `init()` call.

| Attribute                                | Behaviour                              |
| ---------------------------------------- | -------------------------------------- |
| `data-modal-trigger="<id>"`              | Opens the modal with that id on click. |
| `data-modal-close`                       | Closes the nearest enclosing modal.    |
| `data-dropdown` + `data-dropdown-toggle` | Wires a click-to-open dropdown.        |
| `data-tabs` + `data-tabs-tab`            | Tab-switcher, keyboard navigable.      |
| `data-toggle-pressed`                    | Toggles `aria-pressed` + label.        |
| `data-tip`                               | Hover/focus tooltip.                   |

## Versioning

The CDN URL pins to the latest minor:
`/uikit/v1/styles.min.css`. For frozen prototypes, pin a patch:
`/uikit/v1.0.4/styles.min.css`.
