Buttons trigger immediate actions. Use a link for navigation, a button for
anything that changes state on the current page.

<PlaygroundLink
  code={`import { Button } from '@freecodecamp/uikit/primitives';

## Keyboard

| Key           | Action                                     |
| ------------- | ------------------------------------------ |
| Space / Enter | Activates the button.                      |
| Tab           | Moves focus to the next focusable element. |

## Accessibility

Renders a native `<button>` element. The focus ring is 3 px solid
`--blue-mid` — the same ring every interactive primitive uses.
Disabled buttons forward `aria-disabled="true"` and drop out of the
tab order.

## Do & Don't

<DoDont>
  <Fragment slot='do'>
    <ul>
      <li>Use one `primary` button per view to anchor the main action.</li>
      <li>Pair `secondary` with `primary` for cancel / save decisions.</li>
      <li>Write the action verb on the button — never the noun.</li>
    </ul>
  </Fragment>
  <Fragment slot='dont'>
    <ul>
      <li>Don't stack more than one `primary` in a row.</li>
      <li>Don't use `danger` for "delete" if the action is reversible.</li>
      <li>Don't swap Buttons for Links — each carries a different contract.</li>
    </ul>
  </Fragment>
</DoDont>
