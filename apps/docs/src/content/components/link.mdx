`<Link>` is the styled anchor. It forwards every native anchor attribute
and adds a `block` boolean so the anchor can own an entire row (for
navigation menus, card-wide clicks).

## Usage

```tsx
import { Link } from '@freecodecamp/uikit/primitives';

<Link href='/guides' block>
  Start the intro guide →
</Link>;
```
