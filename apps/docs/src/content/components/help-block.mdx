`<HelpBlock>` is the tiny explanatory line that sits under a control. The
`variant` prop flips its tone for success or error without changing the
DOM shape — so `aria-describedby` wiring stays stable.

## Usage

```tsx
import { HelpBlock } from '@freecodecamp/uikit/forms';

<HelpBlock variant='error'>That name is already taken.</HelpBlock>;
```
