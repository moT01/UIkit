`<ToggleButton>` is a pressable button that surfaces its state through
`aria-pressed`. It's the right control for icon-bar toolbars (bold / italic
/ strike) and settings where the label never changes but the meaning flips.

## Usage

```tsx
import { ToggleButton } from '@freecodecamp/uikit/forms';

<ToggleButton
  size='md'
  defaultPressed={false}
  onPressedChange={next => console.log('pressed?', next)}
>
  Bold
</ToggleButton>;
```
