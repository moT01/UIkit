`<Dropdown>` is the menu-button compound: a trigger (`Dropdown.Toggle`),
a panel (`Dropdown.Menu`), and items (`Dropdown.Item`). It owns the
keyboard navigation and the click-outside dismiss behaviour so consumers
stay declarative.

## Usage

```tsx
import { Dropdown } from '@freecodecamp/uikit/overlays';

<Dropdown>
  <Dropdown.Toggle>Actions</Dropdown.Toggle>
  <Dropdown.Menu>
    <Dropdown.Item onSelect={() => edit()}>Edit</Dropdown.Item>
    <Dropdown.Item onSelect={() => share()}>Share</Dropdown.Item>
  </Dropdown.Menu>
</Dropdown>;
```
