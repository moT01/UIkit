Square, flat, high-contrast — the Command-line Chic avatar. Fall back
to initials when no image is available (the 99% case in new sign-ups)
and attach a status dot when presence matters.

## Props

<PropTable
  rows={[
    {
      name: 'name',

      required: true,
      description: 'Display name. Used for `alt` text and initials fallback.'

    {
      name: 'src',
      type: 'string',
      description: 'Image URL. Omit to render initials.'

      name: 'size',

    }
  ]}
/>

## Accessibility

When `src` is present, `alt` is filled from `name`. When falling back
to initials, the wrapper carries `aria-label={name}` and the initials
node is `aria-hidden`. Status dots include an `aria-label`
describing the state.

## Do & Don't

<DoDont>
  <Fragment slot='do'>
    <ul>
      <li>Use `name` even when you have a `src` — it's the alt text.</li>
      <li>
        Keep the initials path working — not every user has uploaded a photo.
      </li>
    </ul>
  </Fragment>
  <Fragment slot='dont'>
    <ul>
      <li>
        Don't skip the border-radius override — Avatars are square by design.
      </li>
      <li>
        Don't layer status dots on top of decorative icons; use them for
        presence only.
      </li>
    </ul>
  </Fragment>
</DoDont>
