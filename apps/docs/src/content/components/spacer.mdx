`<Spacer>` is the explicit layout gap. It's handy when the surrounding
components should not carry margin of their own — typical in MDX content
where paragraphs and code blocks already own their spacing.

## Usage

```tsx
import { Spacer } from '@freecodecamp/uikit/primitives';

<Spacer size={4} />
<Spacer size={20} axis='horizontal' />
```
