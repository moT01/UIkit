`<Tabs>` + `<Tab>` render a proper `role='tablist'` / `role='tab'` /
`role='tabpanel'` triad. Arrow keys move focus between tabs; Enter and
Space activate the focused tab. Works controlled via `activeKey` or
uncontrolled via `defaultActiveKey`.

## Usage

```tsx
import { Tabs, Tab } from '@freecodecamp/uikit/navigation';

<Tabs defaultActiveKey='one'>
  <Tab eventKey='one' title='One'>
    First panel body.
  </Tab>
  <Tab eventKey='two' title='Two'>
    Second panel body.
  </Tab>
</Tabs>;
```
