`<CloseButton>` wraps a native `<button>` with the shared dismiss visual
and a sensible default `aria-label`. It's the right primitive for toast
close-affordances, modal close-×, and dismissible banners.

## Usage

```tsx
import { CloseButton } from '@freecodecamp/uikit/primitives';

<CloseButton onClick={() => setOpen(false)} />;
```
