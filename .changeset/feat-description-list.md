Add `<DescriptionList>` data-display primitive — semantic dl/dt/dd
rendered from an `items` array of `{ term, detail }` pairs. Two
layouts: `vertical` (default) is a two-column grid sized for metadata
panels; `inline` flows term/detail pairs inline for compact status
strips. Terms render as mono terminal eyebrows; details as Lato body.
