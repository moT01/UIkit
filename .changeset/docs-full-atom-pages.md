Ship dedicated MDX pages for every component the storybook covers:

- **Full pages (3C.7)** for the 11 stable atoms — `button`, `alert`,
  `callout`, `badge`, `card`, `panel`, `input`, `checkbox`, `switch`,
  `modal`, and `tooltip`. Each carries prop tables, keyboard notes,
  accessibility guidance, Do/Don't, token chips, and cross-links. The
  `modal` page closes the long-standing Wave 2F dialog-demo gap.
- **Beta stubs (3C.8)** for 9 less-touched components — `toggle-button`,
  `close-button`, `link`, `spacer`, `form-group`, `form-control`,
  `help-block`, `dropdown`, and `tabs`. Stubs carry a usage snippet and
  a link to `/showcase#<id>` so sidebar entries never 404.

Docs-only change, no package bump.
