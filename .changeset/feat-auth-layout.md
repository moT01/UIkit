Add `<AuthLayout>` — centered-card shell for login, signup, forgot,
and magic-link routes. Renders a native `<main>` card with optional
`brand` slot above, optional `footer` slot below, and an opt-in
`pattern` modifier that swaps the plain background for a muted grid
texture (built from CSS gradients against the tertiary token — no
images, no runtime cost).
